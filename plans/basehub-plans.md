# BaseHub Plans

BaseHub offers three tiers: a free Personal plan and two paid plans (Team and Unlimited). All plans include access to the GraphQL API, SDK, webhooks, multiplayer editing, AI chat, analytics, edit mode, live preview, and the command palette. An annual billing discount of 20% applies to paid plans.

## Personal (Free)

- Price: $0/month
- 5,000 blocks across all repos
- 75,000 API requests/month
- 25 GB asset storage
- 1M asset requests/month
- Search and analytics (free during beta)
- Open Graph generator
- Webhooks
- SDK access

## Team

- Price: $15/user/month (billed monthly); ~$12/user/month (billed annually)
- Unlimited blocks
- 500,000 API requests/month
- 500 GB asset storage
- 5M asset requests/month
- All Personal features
- Multiplayer editing
- Diff view
- Templates

## Unlimited

- Price: $350/month (billed monthly); ~$280/month (billed annually)
- Unlimited users
- 5,000,000 API requests/month
- 5 TB asset storage
- All Team features
- Priority support

## Overage Pricing

| Resource | Overage Rate |
|---|---|
| Blocks (beyond free tier) | $1 per 500 blocks |
| API requests (Personal) | $1 per 50,000 requests |
| API requests (Team/Unlimited) | $1 per 100,000 requests |
| Asset storage | $0.10 per GB |
| Asset requests | $1 per 1M requests |
| Search/analytics queries (Personal) | $1 per 10,000 queries |
| Search/analytics queries (Team/Unlimited) | $1 per 25,000 queries |
| Emails | $1 per 1,000 emails |

Source: https://basehub.com/pricing
